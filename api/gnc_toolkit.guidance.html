Computes consensus control for a distributed multi-agent system based on the network Laplacian matrix.

u_i = -K * SUM(a_ij * (x_i - x_j)) In matrix form: U = -K * (L @ X)

Parameters:

  • laplacian_matrix (np.ndarray) – N x N graph Laplacian matrix.

  • gains (np.ndarray) – Gains to apply to the consensus error.

Returns:

np.ndarray

Return type:

N x M array of control inputs.
