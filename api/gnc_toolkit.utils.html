  • ecc (float) – Eccentricity

  • incl (float) – Inclination [rad]

  • raan (float) – RAAN [rad]

  • argp (float) – Argument of perigee [rad]

  • M (float) – Mean anomaly [rad]

Returns:

tuple

Return type:

(a, h, k, p, q, l_mean)
