gravity_gradient_torque(J, r_eci, q_body2eci)[source]

Calculates Gravity Gradient torque in Body Frame. T_gg = 3 * mu / R^5 * (r_body x J * r_body) (Note: r_body = -u_nadir_body * R. Formula using unit vector: 3*mu/R^3 * (u x Ju))
