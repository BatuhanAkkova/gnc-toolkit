gnc_toolkit.guidance.maneuvers.bi_elliptic_transfer(r1: float, r2: float, rb: float, mu: float = 398600.4418) → tuple[float, float, float, float][source]

Calculates the Delta-V requirements and transfer time for a Bi-Elliptic transfer.

Parameters:

  • r1 (float) – Radius of the initial orbit (km).

  • r2 (float) – Radius of the final orbit (km).

  • rb (float) – Radius of the intermediate apogee (km). Must be > max(r1, r2).

  • mu (float) – Gravitational parameter (km^3/s^2). Default is Earth.

Returns:

  • dv1 (float): Delta-V for first burn (km/s).

  • dv2 (float): Delta-V for second burn (km/s).

  • dv3 (float): Delta-V for third burn (km/s).

  • transfer_time (float): Total time of flight (s).

Return type:

tuple[float, float, float, float]

gnc_toolkit.guidance.maneuvers.phasing_maneuver(a: float, T_phase: float, mu: float = 398600.4418) → tuple[float, float][source]

Calculates the Delta-V required for a phasing maneuver to correct a timing error or shift position in the orbit.

Parameters:

  • a (float) – Semi-major axis of the current circular orbit (km).

  • T_phase (float) – Desired time difference to generate (s). Positive to wait (increase period), negative to catch up. Usually T_phase is small relative to orbital period.

  • mu (float) – Gravitational parameter.

Returns:

  • total_dv (float): Total Delta-V (entry + exit) (km/s).

  • t_wait (float): Duration of the phasing orbit (s).

Return type:

tuple[float, float]

Propagates relative state using Clohessy-Wiltshire (Hill’s) equations. Assumes circular target orbit.

Parameters:

  • v0 (np.ndarray) – Initial relative velocity [vx, vy, vz] (km/s).

  • n (float) – Mean motion of target orbit (rad/s).

  • t (float) – Time to propagate (s).

Returns:

  • r_t (np.ndarray): Relative position at time t.

  • v_t (np.ndarray): Relative velocity at time t.

Return type:

tuple[np.ndarray, np.ndarray]
